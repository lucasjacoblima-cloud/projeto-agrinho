# Agro forte 

Foi planejado para servir como um guia prático e interativo. Aqui está o detalhamento de tudo o que estruturamos:
Focado em ensinar como plantar tomate e pimentão de forma sustentável, abordando desde a chocha do container até a rega e luz solar.
Identidade visual (CSS):Um "Eco-tema" moderno com paleta de verdes, sombras suaves e fontes limpas para uma experiência visual agabalada e ecológica.
Um sistema de questões de múltipla escolha desenvolvido para testar o conhecimento do usuário sobre agricultura urbana.
Feedback dinâmico (JS):Lógica que identifica acordes com animações de "bounce" (pulo) e mensagens de incentivo, tornando o aprendizado divertido.
O foco central é mostrar que a produção de alimentos e a preservação do meio ambiente podem caminhar juntas.
